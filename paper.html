Methodology & Data

Data was collected over a 6-month period across controlled sandbox environments and real-world red team engagements. The testbed included Windows 11, Server 2022, and Linux kernels 5.x. Over 1,200 telemetry events were recorded and categorized.

Mitigation Strategies

Based on the identified weaknesses, the following countermeasures are recommended:

  1. Enable PowerShell logging version 5.1+ with script block logging.
  2. Deploy user-mode hook integrity checks at randomized intervals.
  3. Implement network-level DoH filtering via custom rules.
